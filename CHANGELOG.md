# Changelog

All notable changes to Research Companion will be documented in this file.

## [1.0.0] - 2025-01-28

### Added
- Session management for paper searches (no data loss)
- Two search modes: Quick (20 papers) and Comprehensive (500+)
- Input selection and validation for analysis
- Conditional analysis strategies (quick vs comprehensive)
- Citation network analysis (PageRank, betweenness, communities)
- Statistical analysis tools (distributions, frequencies, correlations)
- Full traceability in reports (every claim cites papers)
- Experiment management (unique folders per analysis)
- Search extension and refinement capabilities

### Technical
- Uses Click framework for CLI scripts
- Supports uv for dependency management
- Multi-source search (arXiv, Semantic Scholar, OpenAlex)
- Generic, domain-agnostic analysis scripts
